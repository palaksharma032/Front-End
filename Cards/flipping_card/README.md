# Flipping Card

A simple responsive flipping credit card UI built with HTML and CSS. The card flips on hover to show the back side, making it a clean front-end animation demo.

## Features

- 3D hover flip effect
- Realistic card styling
- Front and back card faces
- Responsive layout for desktop and smaller screens

## Project Structure

```text
flipping_card/
├─ Assets/
│  ├─ chip.png
│  ├─ logo.png
│  └─ logo12.png
├─ flipping_card.html
├─ flipping_card.css
└─ README.md
```

## How to Use

1. Open `flipping_card.html` in a browser.
2. Hover over the card to see the flip animation.
3. Edit the card details or styling in `flipping_card.html` and `flipping_card.css`.

## Notes

- The image folder name is `Assets` with a capital A.
- If you move the files, make sure the image paths in the HTML stay correct.
