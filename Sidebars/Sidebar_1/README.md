# Whimsycore Sidebar

A lightweight, responsive sidebar menu demo built with plain HTML, CSS, and icon CDNs.

Features
- Responsive sidebar with hamburger toggle
- Smooth slide animation and collapse behavior
- Icon-based menu items (Font Awesome + Boxicons)
- Social icons row and scrollable menu when items overflow

Files
- `index.html` — demo page with markup and a small client script
- `style.css` — complete styling and responsive behavior
- `background.jfif` — demo background image used for the preview
- `screenshots/` — example screenshots (replace with real images)

Preview
Open `index.html` in your browser or use the VS Code Live Server extension.

Quick start
1. Clone this repo:

	git clone <repo-url>

2. Open the `Sidebar_1` folder and open `index.html` in a browser.

Notes
- This demo uses Font Awesome and Boxicons via CDNs — an internet connection is required to load icons.
- Replace the files in `screenshots/` with high-resolution PNGs or JPGs for the GitHub repository preview.
