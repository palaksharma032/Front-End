# Quiz Game

A simple browser-based quiz game built with HTML, CSS, and vanilla JavaScript. The game presents multiple-choice questions, tracks the score, and shows a result screen at the end.

## Features

- Start screen with a short intro
- Five multiple-choice trivia questions
- Score tracking during the quiz
- Answer feedback for correct and incorrect choices
- Final result message based on performance
- Responsive layout for smaller screens

## Files

- `index.html` - app structure and screens
- `style.css` - layout, colors, and responsive styling
- `script.js` - quiz data, game logic, and UI updates

## How to Run

1. Open `index.html` in a browser.
2. Click `Start Quiz`.
3. Choose an answer for each question.
4. View your final score and restart if you want to try again.

## Notes

- The project does not require any build tools or external dependencies.
- All logic runs directly in the browser.
