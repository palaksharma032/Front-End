# Login Form 1

A clean, responsive login page built with HTML and CSS. It uses a modern glass-style card layout, layered gradients, and mobile-friendly spacing to create a polished first impression.

## Preview

- Responsive two-panel login layout
- Stylish glassmorphism-inspired card
- Clear input states and call-to-action button
- Works on desktop and mobile screens

## Built With

- HTML5
- CSS3
- Google Fonts: Inter

## Features

- Email or phone input field
- Password input field
- Remember me checkbox
- Forgot password and sign-up links
- Responsive design for smaller screens

## How to Run

1. Open the `Login Form 1` folder in your code editor.
2. Open `Index.html` in a browser.
3. Edit `style.css` to customize the appearance.

## Customization Ideas

- Change the background gradient colors in `style.css`.
- Update the card text to match your brand.
- Replace the input icons with SVGs or Font Awesome icons if you want a different look.
- Connect the form to a backend or authentication service.
